# Hexo-theme-simix

A simple theme for Hexo.

## Installatiion

Execute the following command and modify `theme` in `_config.yml` to `simix`.

```
git clone git://github.com/wuuguo/hexo-theme-simix.git themes/simix
```

## Update

Execute the following command to update.

```
cd themes/simix
git pull
```
